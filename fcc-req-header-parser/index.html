Request Header Parser Microservice

Make an HTTP GET request to /api/whoami on this domain to retrieve information in JSON about your ip address, preferred language, and OS/browser.

Example Request

Input: [current url]/api/whoami

Output: {"ipaddress":"0.0.0.0","language":"en-US,en;q=0.5", "software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}

Timestamp Microservice

Enter a date in either milliseconds since the UNIX epoch or using year, month, and day (month can be number or name). API returns object with milliseconds of entered date from UNIX epoch and a string containing day of the week, day of the month, month, year, and time to represent entered date.

Example Requests

Input: [current url]/api/1454457600000

Output: {"unix":1454457600000,"utc":"Wed, 03 Feb 2016 00:00:00 GMT"}

-------------------------------------

Input: [current url]/api/June 2, 2017

Output: {"unix":1496361600000,"utc":"Fri, 02 Jun 2017 00:00:00 GMT"}
